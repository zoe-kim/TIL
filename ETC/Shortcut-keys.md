# 알아두면 유용한 단축키

단축키의 세계는 무.궁.무.진.해

<br>

## 💡 Chrome
 
* `브라우저 탭`에서 마우스 `휠` 클릭 : 탭 닫기
* `링크`에서 마우스 `휠` 클릭 : 새 탭에서 링크 열기
* `Ctrl` + `Shift` + `T` : 최근 닫았던 탭 열기

<br>

## 💡 Window
 
* `Window` + `Shift` + `S` : 스크린샷

<br>

## 💡 VS Code
 
* `Ctrl` + `/` : // 한 줄 주석
* `Alt` + `Shift` + `A` : /* 블럭 주석 */
* `Ctrl` + `Shift` + `[` : 커서 영역 접기
* `Ctrl` + `Shift` + `]` : 커서 영역 펼치기
* `Ctrl` + `Shift` + `P` 누르고 `약어로 랩핑` 검색 > 감쌀 `태그` 입력 : 상위 태그로 감싸기

* `Ctrl` + `C` : 터미널 서버 종료 (새 입력창 띄움)
